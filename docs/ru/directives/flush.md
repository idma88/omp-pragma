Указывает, что все потоки имеют одинаковое представление памяти для всех общих объектов.

```cpp
#pragma omp flush [(var)]
```

### Параметры

*var*<br/>
(Необязательный) Разделенный запятыми список переменных, представляющих объекты, которые вы хотите синхронизировать. Если *var* не указан, сбрасывается вся память.

### Примечания

Директива `flush` не поддерживает клаузы.
