Позволяет указать, что часть кода должна выполняться в одном потоке, не обязательно в основном потоке.

```cpp
#pragma omp single [clauses]
{
   code_block
}
```

### Параметры

*clauses*<br/>
(Необязательный) Ноль или несколько клауз, см. раздел **Примечания**.

### Примечания

Директива `single` поддерживает следующие клаузы:

- `private`
- `firstprivate`
- `copyprivate`
- `nowait`

Чтобы указать, что часть кода должна выполняться только в основном потоке, используйте директиву `master`.
